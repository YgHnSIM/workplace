# 우체국물류지원단 물류노동조합 공개 자료실 (Workplace)

우체국물류지원단 물류노동조합의 공식 성명서와 공개 자료를 모아둔 저장소입니다.

## 배포 문서
- **물류노동자 노동강도 차별 해결 촉구 성명서 (2026년 7월)**
  - 파일명: [statement/성명서_202607.html](statement/성명서_202607.html) (혹은 [statement/index.html](statement/index.html))
  - 내용: 교섭대표노조와 우체국물류지원단, 우정사업본부에 노동강도 증가 대책 마련 및 임금·상여금 차별 철폐를 촉구하는 내용입니다.
  - 기능: 텍스트 자동 복사, 본문 글꼴 크기 변경, A4 최적화 인쇄 기능을 지원합니다.
- **운영위원회 회의록 공개용 요약본**
  - 파일명: [MoM/index.html](MoM/index.html)
  - 내용: 원본 회의록은 `_source/MoM`에서 관리하고, 공개 페이지에는 회의 개요·주요 결정사항·차기 회의 안내만 자동 생성합니다.
  - 기능: `npm run build`로 회의록 HTML과 목록을 재생성하고, `npm run validate`로 링크와 placeholder를 확인합니다.

## 라이선스
본 자료의 저작권은 우체국물류지원단 물류노동조합에 있으며, 출처 표시 후 자유롭게 공유 및 인용할 수 있습니다.
